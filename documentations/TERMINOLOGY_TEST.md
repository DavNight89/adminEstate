## ANSWER KEY

### Section A Answers (Terms 26-42)

1. **JSX** = JavaScript XML - React's XML-in-JavaScript syntax
2. **JSON** = JavaScript Object Notation - Data interchange format
3. **Fetch API** = Modern browser HTTP request interface
4. **localStorage** = Browser key-value persistent storage
5. **MIME Types** = Media type classification (e.g., text/html, image/png)
6. **URLSearchParams** = URL query string manipulation
7. **HTTP Methods** = GET, POST, PUT, DELETE (request types)
8. **Content-Type Headers** = HTTP header specifying payload format
9. **webpack** = JavaScript module bundler
10. **npm** = Node Package Manager - JavaScript dependency manager
11. **pip** = Python Package Manager - Python dependency manager
12. **ESLint** = JavaScript code linting tool
13. **Autoprefixer** = CSS vendor prefix automation
14. **PostCSS** = CSS transformation tool
15. **Terser** = JavaScript minification tool
16. **Source Maps** = Debug mapping for minified code
17. **Node.js** = JavaScript runtime environment

### Section B Answers (Terms 125-142)

18. **Bento Grid** = Asymmetric grid layout (used for screenshots)
19. **Click-to-Enlarge** = Image expansion interaction
20. **Hover Effects** = Mouse-over visual changes
21. **Zoom Effects** = Image magnification
22. **Matplotlib** = Python plotting library
23. **Seaborn** = Statistical data visualization
24. **ExcelJS** = Excel manipulation (JavaScript library)
25. **Mock Data** = Test/placeholder data
26. **Pagination** = Data page division
27. **Filtering** = Data subset selection
28. **Sorting** = Data ordering
29. **Search Functionality** = Text-based data finding
30. **Error Recovery** = Failure handling mechanism
31. **API Fallback** = Backup when primary API fails
32. **Network Error Handling** = Connection failure management
33. **Persistence Layer** = Data storage abstraction
34. **Local Storage Fallback** = Offline data alternative
35. **Bulk Operations** = Mass data processing

## Study Tips

### For Terms 26-42 (Web Technologies & Build Tools)
- **Web Technologies**: These are fundamental browser APIs you interact with daily
- **Build Tools**: Focus on understanding *what problem each tool solves*
- **Mnemonic for HTTP Methods**: "Get Puppies Please Don't" (GET, POST, PUT, DELETE)

### For Terms 125-142 (Layout/Development/Architecture)
- **Layout Patterns**: Visual/interactive concepts - try to visualize them
- **Python Libraries**: Remember Matplotlib (plots) vs Seaborn (statistical viz)
- **Development Concepts**: Think about *why* each exists (pagination = handle large datasets)
- **Architecture Terms**: These solve data persistence/offline challenges
