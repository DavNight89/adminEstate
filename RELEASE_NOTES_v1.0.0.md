# AdminEstate Desktop Application v1.0.0

## Windows Desktop App (Standalone)

**Download and run AdminEstate offline** - No installation required!

### What's Included:
- ✅ Full property management system
- ✅ Tenant portal
- ✅ Work order management
- ✅ Financial tracking
- ✅ Document management
- ✅ PostgreSQL database support
- ✅ Runs completely offline (data stored locally)

### Installation Instructions:

1. **Download** the ZIP file (`AdminEstate-Windows-x64.zip`)
2. **Extract** the ZIP file to your desired location
3. **Open** the extracted folder and run `AdminEstate.exe`

### System Requirements:
- Windows 10 or later (64-bit)
- 500 MB free disk space
- No additional software required

### Features:
- Property portfolio management
- Tenant tracking and communication
- Maintenance request system
- Financial analytics dashboard (powered by Pandas)
- Document storage and management
- Rent collection tracking
- Offline-first architecture
